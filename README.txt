# Cyber Scam Prevention Application  

**A revolutionary application to combat cyber scams worldwide, with a special focus on scams in India.**  
Our solution involves a mobile application and a web application designed to protect users from fraud during phone calls by detecting keywords related to cyber scams and taking preventive actions automatically.

- **Chatbot Assistance:** A built-in chatbot guides users on how to identify and avoid cyber scams.  
- **Web Application:** A companion web platform to track and analyze scam-related data globally.  
- **Multi-Language Support:** Making it accessible to users worldwide.

## üöÄ How It Works  

1. **Call Monitoring:**  
   The mobile application uses speech recognition technology to analyze ongoing calls in real-time.  

2. **Keyword Detection:**  
   If keywords like "OTP," "password," or "link" are detected, the system flags the call.  

3. **Action:**  
   Based on pre-configured settings, the app either alerts the user or automatically ends the call.  

4. **Education and Reporting:**  
   Users can report suspected scam numbers through the app or web platform.
